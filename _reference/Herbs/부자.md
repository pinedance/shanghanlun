#### 명칭

其品凡七, 本同而末異,
* 〔烏頭〕其初種之母爲烏頭,
* 〔附子〕附烏頭旁生者謂附子,
* 〔鬲子〕又左右附而偶生者謂鬲子,
* 〔天雄〕種而獨生無附, 長三四寸者謂天雄,
* 〔天錐〕附而尖者謂天錐,
* 〔側子〕附而上出者謂側子,
* 〔漏藍子〕附而散生者謂漏藍子


#### 취상

* 〔烏頭〕老陰之生育已竟者也. ... 其中空, 以氣爲用 ... 氣主發散 ... 發散者能外達腠理, 故主中風惡風, 洗洗出汗, 咳逆上氣
* 〔天雄〕孤陽之不能生育者也. ... 其中實, 以精爲用 ... 精主斂藏 ... 斂藏者能內入筋骨, 故主歷節痛, 拘攣緩急, 筋骨不强, 身重不能行步
* 〔附子〕卽烏頭天雄之種, 含陰包陽者也. ... 附子則兼備二氣, 內充實外强健 ...  斯能兼擅二物之長, 其用較二物爲廣矣.
  - 上則風寒咳逆邪氣
  - 中則癥堅積聚血瘕
  - 下則寒濕踒躄拘攣膝痛不能行步

附子味辛烈而氣雄建, 又偏以氣爲用, ... 能引火下歸

#### 補眞陽, 以散壅

補眞陽, 以散壅
* 陽之虛而上浮者, 卽能於極上收之,  _如腎厥頭痛之類_
* 陽之虛而筋節緩機關弛者, 卽能於筋節機關强之堅之,  _如腰脚冷弱之類_

與麻黃ㆍ桂枝ㆍ茯苓ㆍ白朮有異
* 附子之〔治風寒〕, 非直治風寒也, 陽氣不榮, 風寒侵侮, 陽振而風寒自退.
* 附子之〔利關節〕, 非直利關節也, 筋得寒則攣, 得熱則弛, 筋弛而關節自舒.

與甘遂ㆍ大黃有異
* 附子之〔治水〕, 非直治水也, 水寒相搏爲䭇, 是中寒, 非外寒也, 去中寒而水無與搏矣.
* 附子之〔治滿〕, 非直治滿也, 濁氣上則脹, 是陰逆非氣盛也, 陽見晛則陰翳消矣

兼有表證者用生附子, 宜乎合表藥用者, 皆生附子

#### 氣爲寒折, 陰長陽消

惟其`氣爲寒折, 陰長陽消`, 附子遂不容不用矣

汗後下後用附子證
* `無表證而煩躁`, 則附子必須用 _乾薑附子湯_ _茯苓四逆湯_
* `下後陰盛`, 不論上衝下洩, 皆須用附子 _桂枝去芍藥湯_ _四逆湯_
* `汗後惡風惡寒不罷`者, 舍附子無能爲力 _桂枝加附子湯_ _芍藥甘草附子湯_ _眞武湯_

未經汗下用附子證, 其機在於脈沈微 （陰氣自盛於內）
* 陰氣盛而陽自困 _小靑龍去麻黃加附子湯_ _麻黃附子細辛湯_ _麻黃附子甘草湯_
* 陰濕盛而困陽 _桂枝附子湯_ _白朮附子湯_ _甘草附子湯_

#### 병증 분석 : 身體四肢及骨節疼痛 > 惡寒，腹痛

主逐水也。

故能治[惡寒]( {{site.sympurl}}/오한 )，身體四肢及骨節疼痛，或[沈重]( {{site.sympurl}}/침중 )，或[不仁]( {{site.sympurl}}/불인 )，或[厥冷]( {{site.sympurl}}/궐냉 )。

## 임상메모

### 주요 증상

심한 惡寒
* 환자가 주소로 호소할 정도의 강도（몸을 따뜻하게 하여 해소되지 않는）
* 궐냉의 경우 타각적 확인 가능할 정도 _厥冷_

심한 疼痛（沈重、不仁、厥冷）
* 관절가동범위 제한을 동반하거나 _拘急_
  - _"새벽에 손가락이 뻣뻣하고 잘 안움직여요"_
* 침중(몸이 무겁다)을 동반하는 경우 _沈重_
  - _"몸이 물에 젓은 솜처럼 무거워요"_
* 이상감각을 동반하는 경우 많음 _不仁_
  - _"이 곳이 내 살이 아닌 것 같아요"_
